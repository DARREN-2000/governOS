# IntentGraph — Product Requirements Document

## Vision

IntentGraph compiles human intent into trusted software.

## Problem

The market is fragmented. No single product combines:

- Personal + enterprise + developer use cases
- Open standards (MCP, A2A)
- Local/private execution options
- First-class preview/approval/rollback as the main UX

## Solution

A universal action OS that turns natural-language goals into trusted, reusable workflows.

## User Personas

### Personal Users

- Track bills and subscriptions
- Organize documents
- Schedule and manage appointments
- Fill forms and handle admin tasks

### Teams & Companies

- Employee onboarding/offboarding
- Support escalation workflows
- Invoice and procurement automation
- Incident response playbooks
- Compliance evidence gathering

### Developers

- Build custom action packs
- Automate issue-to-PR workflows
- Create incident runbooks
- Publish reusable workflow templates

## v1 Scope

### Surfaces

- Web app
- Browser extension
- Slack bot
- GitHub app

### Connectors

- Gmail / Outlook
- Google Calendar / Outlook Calendar
- Google Drive / OneDrive
- Slack
- GitHub
- Notion or Jira
- Browser automation (fallback only)

### Killer Workflows

1. Email triage + follow-up drafting
2. Meeting summary → tasks → Slack update
3. Bug issue → reproduce → failing test → draft PR
4. New employee onboarding checklist
5. Recurring bill tracker + receipt filing

### Hard Rules

- No side-effecting action without preview
- No deletion, provisioning, money movement, or external sending without approval
- Prefer direct API integrations first
- Browser/computer-use only when there is no API path

## Business Model

- Free personal tier
- Pro personal tier (more connectors/runs)
- Team tier (per seat + per workflow usage)
- Enterprise tier (private deployment, governance, audit)
- Marketplace revenue share for premium action packs

## Success Metrics

- Monthly active workflows
- Workflow completion rate
- Time saved per user per week
- Template reuse rate
- Connector adoption rate
