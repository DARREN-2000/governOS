# Frequently Asked Questions

## General

**What is the difference between an Intent and a Plan?**
An Intent is the user's natural language goal. A Plan is the deterministic, step-by-step technical specification generated by GovernOS to achieve that goal.

**Can I write my own Action Plugins?**
Yes! See the SDK Guide for how to implement the `preview()`, `execute()`, and `compensate()` interface for custom integrations.

## Execution

**What happens if a step in a multi-step plan fails?**
GovernOS relies on the `compensate()` method of action plugins. If step 3 fails, it will attempt to run `compensate()` on step 2, and then step 1, to roll the system back to its original state.

**Why is my intent stuck in the 'waiting_approval' state?**
The Policy Engine flagged the plan as requiring human review. An Administrator must approve it via the Web Dashboard or API before execution can proceed.
